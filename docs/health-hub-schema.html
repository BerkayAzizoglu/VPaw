Core Health Entities
Vet Visit Anchor
Her sey buraya baglana da bilir, baglanmak zorunda da degil
  • visitDate: string (ISO)
  • reasonCategory: checkup | vaccine | illness | injury | surgery | emergency | follow_up
  • status: completed | planned | canceled
  • clinicName?: string
  • vetName?: string
  • followUpDate?: string
  • expense?: { amount, currency }
  • notes?: string
  • linkedRecordIds?: string[] ← otomatik
Medical Record Core
Tek tip: vaccine, condition, surgery, medication, lab hepsi bu
  • type: vaccine | condition | surgery | medication | lab_result
  • title: string (dropdown'dan)
  • eventDate: string (ISO)
  • status?: active | resolved | monitoring | completed | upcoming
  • vetVisitId?: string ← opsiyonel link
  • subcategory?: string (spesifik isim)
  • dueDate?: string (sonraki tarih)
  • valueNumber? + valueUnit? (lab icin)
  • expense?: { amount, currency }
  • note?: string
  • createdAt, updatedAt
Reminder Auto-link
Kayitlardan otomatik + manuel olusturulabilir
  • type: medical | care
  • subtype: vaccine | vet_visit | medication | food | litter | walk | custom
  • scheduledAt: string
  • frequency: once | daily | weekly | monthly | custom
  • sourceRecordId?: string ← hangi kayittan geldi
  • sourceType?: medical_record | vet_visit | manual
  • completedAt?: string
  • isActive: boolean
Weight Entry
Profildeki kilo takibi - ayri kalir
  • value: number
  • unit: kg | lb
  • date: string (ISO)
  • vetVisitId?: string
Expense New
Her kayda opsiyonel olarak eklenir, ayri entity degil
  • amount: number
  • currency: TRY | USD | EUR
  • category: auto (parent record type'tan)
  • date: auto (parent record date)
Document Future
Dosya/foto ekleme - gelecekte
  • name: string
  • uri: string
  • type: photo | pdf | lab_report
  • parentRecordId: string

✨ Premium AI Health Summary

Structured data = Accurate AI. Her kategori dropdown'dan secildigi icin, AI tam olarak anlayabilir.

Health Score
Tum verilere bakarak 0-100 arasi saglik skoru hesaplar. Aktif condition sayisi, asi durumu, kilo trendi, ilac uyumu
Risk Alerts
"Diyabet + kilo artisi = risk" gibi cross-category analizler. Irksa yatkinliklar
Expense Forecast
Gecmis harcamalara dayanarak yillik maliyet tahmini. Kategori bazli breakdown
Care Timeline
"Nisan: DHPP booster, Mayis: dis temizligi, Haziran: kan tahlili" gibi gelecek plan
AI'nin Ihtiyac Duydugu Structured Data
  • Conditions: type=condition, status (active/resolved), name (dropdown)
  • Vaccines: type=vaccine, name (dropdown), dueDate
  • Medications: type=medication, name, frequency, status (active)
  • Lab Results: type=lab_result, valueNumber, valueUnit, status (normal/abnormal)
  • Weight: value array + dates → trend analysis
  • Expenses: amount + currency per record → category breakdown
  • Pet Profile: breed, age, sex → breed-specific risk factors
Free-text vs Structured Karsilastirmasi
Alan Eski (Free-text) Yeni (Structured) AI Farki
Vaccine name "rabis asisi yaptirdim" Dropdown: "Rabies" AI ne asi oldugunu %100 bilir
Condition "bobrek sorunu varmis" Dropdown: "Bobrek Yetmezligi" + status: active AI tedavi onerebilir, risk hesaplar
Medication "ilac verdiler gunde 2" Name: "Metacam" + frequency: twice_daily + status: active AI etkilesim kontrolu + uyum takibi
Expense (yok) amount: 350, currency: TRY AI yillik maliyet tahmini yapabilir
Lab value "kan degerleri normalmis" valueNumber: 45, valueUnit: "mg/dL", status: normal AI zaman icinde trend analizi yapabilir
