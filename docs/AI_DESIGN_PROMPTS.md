# AI Design Prompt Commands

Use these prompts when you want AI-generated UI to follow the project's `DESIGN.md`.

## 1) New screen
"Use `DESIGN.md` as the design authority. Create a new React Native screen named `XScreen` for VPaw. Keep spacing, typography, colors, corner radii, and component styling aligned with DESIGN.md. Reuse existing components where possible."

## 2) Existing screen restyle
"Refactor `screens/XScreen.tsx` to match `DESIGN.md` exactly. Do not change business logic; only update layout and visual styling."

## 3) Component creation
"Create `components/XCard.tsx` based on `DESIGN.md`. Include states (default, pressed, disabled) and keep API simple."

## 4) Strict mode (for consistency)
"Apply `DESIGN.md` strictly. Avoid introducing new colors, font sizes, spacings, or radii unless required by accessibility."

"`DESIGN.md` dosyasýný ana tasarým kaynađý kabul et. Oluţturacađýn ekran/bileţenlerde bu dosyadaki tipografi, renk, boţluk ve bileţen stil kurallarýna birebir uy."
